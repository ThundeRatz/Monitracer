# New MoniTracer

## Installations

Quick overview about the installation of necessary programs for the development is [here](docs)

## Running 

To update the node modules, you need to run to folowing command in the repository's root:

```bash
npm install 
```

Then, you can run react native:

```bash
npx react-native start 
```

On another terminal you can begin run the simulation:

```bash
npx react-native run-android 
```
